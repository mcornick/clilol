## clilol get weblog

get a weblog entry

### Synopsis

Gets one of your weblog entries by ID.

Specify the ID with the --id flag.

```
clilol get weblog [flags]
```

### Options

```
  -h, --help        help for weblog
  -i, --id string   ID of the entry to get
```
