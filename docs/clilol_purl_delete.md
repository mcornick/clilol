## clilol purl delete

delete a PURL

### Synopsis

Deletes a PURL.

Specify the PURL name with the --name flag.

Note that you won't be asked to confirm deletion.
Be sure you know what you're doing.

```
clilol purl delete [flags]
```

### Options

```
  -h, --help          help for delete
  -n, --name string   Name of the PURL
```
