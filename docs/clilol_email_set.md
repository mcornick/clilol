## clilol email set

set email forwarding address(es)

### Synopsis

Sets your email forwarding address(es).
	
To specify multiple addresses, separate them with commas.

```
clilol email set [flags]
```

### Options

```
  -d, --destination string   address(es) to forward to
  -h, --help                 help for set
```
