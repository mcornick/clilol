## clilol create purl

Create a PURL

### Synopsis

Creates a PURL.

The PURL will be created as unlisted by default. To create a listed
PURL, use the --listed flag.


```
clilol create purl [name] [url] [flags]
```

### Options

```
  -h, --help     help for purl
  -l, --listed   create as listed (default false)
```
