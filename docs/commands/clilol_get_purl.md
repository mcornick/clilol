Get a PURL

## Synopsis

Gets a PURL by name.

The address can be specified with the --address flag. If not set,
it defaults to your own address.

```
clilol get purl [name] [flags]
```

## Options

```
  -a, --address string   address whose PURL to get
  -h, --help             help for purl
```
