## clilol get dns

Get a DNS record

### Synopsis

Gets a DNS record by attributes.

```
clilol get dns [name] [type] [data] [flags]
```

### Options

```
  -h, --help           help for dns
  -p, --priority int   priority of the DNS record
  -T, --ttl int        time to live of the DNS record (default 3600)
```
